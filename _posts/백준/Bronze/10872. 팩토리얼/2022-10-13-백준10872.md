### 문제 설명

<p>0보다 크거나 같은 정수 N이 주어진다. 이때, N!을 출력하는 프로그램을 작성하시오.</p>

### 입력 

 <p>첫째 줄에 정수 N(0 ≤ N ≤ 12)이 주어진다.</p>

### 출력 

 <p>첫째 줄에 N!을 출력한다.</p>

### 정답 코드 

```py
def fact(N):
    if N <=1:
        return 1
    return N*fact(N-1)

N = int(input())
print(fact(N))
```
