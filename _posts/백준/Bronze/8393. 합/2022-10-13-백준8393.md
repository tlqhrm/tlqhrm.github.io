### 문제 설명

<p>n이 주어졌을 때, 1부터 n까지 합을 구하는 프로그램을 작성하시오.</p>

### 입력 

 <p>첫째 줄에 n (1 ≤ n ≤ 10,000)이 주어진다.</p>

### 출력 

 <p>1부터 n까지 합을 출력한다.</p>

### 정답 코드 

```py
n = int(input())

x = 0
for i in range(1,n+1):
    x += i
print(x)

```
